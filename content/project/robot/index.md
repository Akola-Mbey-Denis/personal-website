title: TelloSLAM
summary: This project is an implementation of  SLAM package for the Tello Robot using ORBSLAM2 algorithm. ORBSLAM2 is near real-time online visual SLAM algorithm that enables a robot to to build a map of unknown environment and to simultaneously determine its location within the map. This SLAM algorithm adopted and used to implement a ROS package for the Tello robot.
